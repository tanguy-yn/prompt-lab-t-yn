# Checklist d'audit

Référence détaillée chargée à la demande. Critères précis par axe + checklists spécifiques aux livrables marketing.

## Barème /5 par axe

### 1. Clarté de l'objectif

| Note | Critère |
|---|---|
| 5 | Livrable précis, mesurable, et un seul objectif principal |
| 4 | Livrable clair mais 1 ambiguïté résiduelle (longueur, format, etc.) |
| 3 | Objectif compréhensible mais non mesurable |
| 2 | Objectif vague, sujet à interprétation |
| 1 | Plusieurs objectifs en compétition, sans priorité |
| 0 | Aucun objectif identifiable |

### 2. Contexte & persona

| Note | Critère |
|---|---|
| 5 | Persona du modèle + audience cible précisément définis |
| 4 | Persona OU audience précis, l'autre implicite mais plausible |
| 3 | Persona générique, audience large |
| 2 | Persona absent, audience devinable |
| 1 | Aucun ancrage |
| 0 | Pas de contexte du tout |

### 3. Format de sortie

| Note | Critère |
|---|---|
| 5 | Structure exacte (titres, sections, longueur, type) |
| 4 | Structure majoritairement spécifiée, 1 zone floue |
| 3 | Format général indiqué (article, post, email) sans détail |
| 2 | Format mentionné mais sans contraintes structurelles |
| 1 | Format implicite |
| 0 | Aucune indication |

### 4. Contraintes

| Note | Critère |
|---|---|
| 5 | Ton + style + exclusions + mots-clés explicites |
| 4 | 3 dimensions sur 4 couvertes |
| 3 | 2 dimensions couvertes |
| 2 | 1 seule dimension |
| 1 | Contraintes mentionnées mais inactionnables |
| 0 | Aucune |

### 5. Activabilité marketing

| Note | Critère |
|---|---|
| 5 | Sortie directement publiable sans retouche |
| 4 | Sortie utilisable après relecture rapide |
| 3 | Sortie nécessite édition substantielle mais structure ok |
| 2 | Sortie requiert refonte partielle |
| 1 | Sortie inutilisable en l'état |
| 0 | Le prompt ne peut pas produire de livrable marketing |

## Checklists par livrable

### Article SEO

- [ ] Mot-clé principal explicite
- [ ] Mots-clés secondaires listés
- [ ] Volume cible (nombre de mots)
- [ ] Title tag + meta description demandés
- [ ] Structure H1/H2/H3 spécifiée
- [ ] FAQ ou tableau récapitulatif demandé si pertinent
- [ ] Audience et intention de recherche définies
- [ ] Référence éditoriale citée

### Post LinkedIn

- [ ] Hook spécifié (chiffre, contre-intuition, mini-histoire)
- [ ] Longueur en caractères
- [ ] Voix de l'émetteur définie (fondateur, expert, etc.)
- [ ] CTA implicite ou explicite
- [ ] Liste « éviter » présente (emojis décoratifs, hashtags, formules creuses)
- [ ] Objectif d'engagement précisé (commentaires vs likes vs reach)

### Carrousel Instagram

- [ ] Nombre de slides
- [ ] Mots max par slide
- [ ] Structure narrative (hook / développement / récap / CTA)
- [ ] Caption demandée séparément
- [ ] Hashtags : nombre et type
- [ ] Objectif (saves, partages, follows)

### Prompt image (Midjourney/DALL-E)

- [ ] Sujet principal précis
- [ ] Style (photo / illustration / 3D)
- [ ] Lumière et ambiance
- [ ] Composition
- [ ] Palette ou référence
- [ ] Negative prompt
- [ ] Paramètres MJ (ratio, version, stylize)
- [ ] Format final attendu

### Newsletter

- [ ] Subject line + preview text demandés
- [ ] Longueur cible
- [ ] Une seule idée centrale ?
- [ ] CTA principal unique
- [ ] Voix éditoriale référencée
- [ ] Métrique cible (open rate, CTR)
- [ ] PS prévu

### Copywriting (landing, ad, email)

- [ ] Promesse principale claire
- [ ] Awareness level de l'audience
- [ ] Preuves (proof points) listées
- [ ] CTA unique
- [ ] Plateforme cible précisée
- [ ] Ton de marque référencé
- [ ] Longueur ou nombre de variations

## Drapeaux rouges (pénalisent au moins 2 axes)

- Le prompt commence par « Tu es un expert en... » sans rien d'autre
- Aucun verbe d'action impératif
- Demande contradictoire (ex : « court mais détaillé »)
- Plus de 3 livrables différents demandés en un seul prompt
- Aucune référence concrète (style, marque, auteur)
- Pas un seul exemple ni anti-exemple
